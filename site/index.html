🅰 LOCAL 부문 심사 기준 (모두의 창업 공식)

경영 전문가·현직 투자자로 구성된 책임멘토단이 다음 3축을 종합 평가:

s1
차별성
기존 서비스·제품과 무엇이 다른가?
s2 ★ 핵심
지역가치
지역 자원·특색을 어떻게 활용하는가?
s3
성장 가능성
지역에 뿌리 내리고 더 큰 시장으로 뻗을 수 있나?

🅱 TECH 부문 심사 기준 (정부 표준 추정)

공식 기준 미공개 → 한국 정부 창업지원사업(예비창업패키지·초기창업패키지·팁스) 표준 기준 추정 적용:

s1
차별성·혁신성
기존 대비 기술 차별점 (단순 GPT wrapper는 감점)
s2
사업성·시장성
명확한 고객·수익모델·TAM이 있는가?
s3
성장 가능성
확장성·SaaS 단위경제·글로벌 잠재력

🎯 통과 예측 산출 방식

  1. 블라인딩: 닉네임·좋아요·등록 시각·분야 태그 모두 제거. id + 한 줄 요약만 LLM에 노출. (검증: 좋아요-점수 상관 r ≈ 0.04, 거의 0)
  2. 3축 스코어링: 각 축 1-10 정수, 평균 5-6 캘리브레이션. 10개 LLM 서브에이전트 병렬 실행.
  3. 합산 + 분야 내 순위: total = s1+s2+s3 (3-30점). 분야(TECH/LOCAL) 내에서만 순위 매김.
  4. 시나리오 컷오프: 분야별 가정한 통과율(예: TECH 50%) → 상위 N명을 통과 예측.
    • 낙관 (82% / 94%): 최종 ~6,000명 가정 (현재 추세)
    • 중도 (50% / 60%): 최종 ~10,000명 가정
    • 비관 (30% / 40%): 최종 ~16,000명 가정 (예비창업패키지 50:1 참고)
  5. Lock: 예측 파일 SHA256 해시 기록 → 발표 후 변경 불가능 증명.
  6. 발표 후 평가: Precision/Recall/AUROC/Calibration plot 산출 예정.
⚠ 한계: 한 줄 요약만 사용 (심사위원은 본문·팀·영상까지 봄). 단일 스코어링 패스. 비공개 풀(~37%) 보이지 않음. TECH 기준은 추정. 예측 천장이 본질적으로 낮음을 인지.
